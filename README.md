## 安装程序

安装包在两个文件夹内

```
CRCVerifyTool_Winform版
	Release
		setup.exe
		CRCVerifyTool_Winform版.msi
CRCVerifyTool_WPF版
	Release
		setup.exe
		CRCVerifyTool_WPF版.msi
```
